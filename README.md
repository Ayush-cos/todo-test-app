# Redux Toolkit Todo Demo

Simple demo app to learn Redux Toolkit patterns modeled after the project's `help.md`.

Run locally:

```bash
npm install
npm run dev
```

What this includes:
- Vite + React + TypeScript scaffold
- `src/store/slices/todos` demonstrating:
  - `interface.ts`, `requests.ts` (thunks with cancel tokens + condition),
    `extra-reducers.ts`, `selectors.ts`, `index.ts`
- Minimal UI to add & list todos
